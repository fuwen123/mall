退换货单手册

操作提示

  • 买家提交的退货申请, 商家同意并平台审核通过之后, 以余额的方式返回到用户账户上.
  • 当审核通过以后，收取到客户的货物，再进行下一步发货操作
  • 在同意审核以后直接就可以退款给客户（注意：在同意用户退款退款之后，请先确认是否收到用户退还的货物，再进行下一步退款操作）
  • 支付原路退回：用户如何支付，就如何退还给到用户。
  • 退款到用户余额：无论用户选择何种支付方式，都是退款到用户商城账户里。
