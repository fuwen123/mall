操作提示

  • 预售管理, 由平台设置管理.
  • 预售活动，预售简而言就是先付定金，在尾款支付时间内支付掉尾款
  • 活动成功：当到达尾款支付时间以后，用户支付尾款，预售活动成功完成。
  • 活动失败：退还用户定金，并且下架该预售商品，标记活动失败。
